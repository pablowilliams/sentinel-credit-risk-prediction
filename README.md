# Project SENTINEL: Credit Risk Prediction

Machine learning ensemble for consumer loan default prediction.

## Overview
SENTINEL uses gradient boosting methods to predict loan defaults, achieving AUC-ROC of 0.89 on held-out test data.

## Dataset
Lending Club loan data (15,000 records, 25 features) from Kaggle.

## Structure
- `src/` - Model training and evaluation code
- `data/` - Kaggle dataset (lending_club_loans.csv)
- `figures/` - Visualisations
- `docs/` - Technical report
